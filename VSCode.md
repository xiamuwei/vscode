# VSCode

+ vscode alt+<- 返回键是返回之前查看的代码处 ； alt+-> 反转上面操作

+ alt+f12 迅速查看函数定义



+ 折叠代码
  + vscode 折叠所有区域代码的快捷键： ctrl +k , ctrl+0
        先按下ctrl +K ， 再按下ctrl + 0 (这个是零
  + 展开所有折叠区域代码的快捷键： ctrl + k, crtl +0 
        先按下ctrl + K，再按下ctrl + J
